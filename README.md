# Admin_Panel
Modern admin panel packages including basic, standard and pro versions built with Next.js and Firebase.

## Repository structure

Each package is a standalone Next.js project and can be installed, developed and deployed independently.

```text
Admin_Panel/
├── temel-paket/
├── standart-paket/
└── pro-paket/
```

## Packages

- `temel-paket`: Basic admin panel package.
- `standart-paket`: Professional admin panel package.
- `pro-paket`: Advanced admin panel package.

## Development

Run commands inside the package you want to work on:

```bash
npm install
npm run dev
```
